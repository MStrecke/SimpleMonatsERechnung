SimpleMonatsERechnung
=====================

# Target audience

This utility is intended for a German audience. Due to a newly adopted German law nearly EVERYONE who writes invoices is obliged to create electronic invoices by 2025 or 2028.

The program demonstrates how to create such an invoice with a minimal data set and how to extend the Mustang library when its functionality is lacking.

If there is a larger international interest, I will translate the German documentation.

# Zielpublikum

Mit dem Wachstumschancengesetz ist ab 2025 und spätestens ab 2028 **jeder** Gewerbetreibende und Freiberufler, der Rechnungen mit Umsatzsteuer schreibt, gesetzlich verpflichtet, eine e-Rechnung zu erstellen.

Dieses Programm zeigt, wie man dies mit möglichst wenig Daten schafft, und wie man die Mustang-Bibliothek erweitert, wenn einem bestimmte Funktionen fehlen.

Die vollständige Übersicht finden Sie in der Datei [liesmich.md](liesmich.md).
